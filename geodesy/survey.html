  • Mapping Software
    • GeoCommunity -- Free Software (a directory of free software tools and utilities in geomatics)
      • Map viewers
      • Web mapping
      • Contouring/3D
      • Plotting/printing
    • GMT: Generic Mapping Tools (a free open source collection of ~60 tools for manipulating geographic and Cartesian data sets (including filtering, trend fitting, gridding, projecting, etc.) and producing Encapsulated PostScript File (EPS) illustrations ranging from simple x-y plots through contour maps to artificially illuminated surfaces and 3-D perspective views. GMT supports ~30 map projections and transformations and comes with support data such as coastlines, rivers, and political boundaries)
    • MATLAB Mapping Toolbox (commercial; a comprehensive set of functions and graphical user interfaces for building map displays and performing geospatial data analysis in MATLAB, supporting key mapping and geospatial data analysis, manipulation, and visualization tasks for applications such as earth and planetary scientific research, oil and gas exploration, environmental monitoring, insurance risk management, aerospace, defense, and security)
    • M_Map (a free mapping toolbox for MATLAB)
