
CDF SOFTWARE ARCHIVE
-----------------------------
Last revised 22 November 2000

This directory contains files for program CDF for computing CDF
(Cummulative Density Functions) for various probability distributions.
Based on the public domain statistics library DCDFLIB v1.1 (Nov 97).


DIRECTORY CONTENTS
------------------
README.TXT     This file.

cdf11lnx.zip   Compressed ZIP file of CDF version 1.1 for HP-UX
               (Unix). Contains executable binary, Fortran source code
               and documentation. Use forinstallation on other Unix
               systems.

cdf11unx.zip   Compressed ZIP file of CDF version 1.1 for HP-UX
               (Unix). Contains executable binary, Fortran source code
               and documentation. Use forinstallation on other Unix
               systems.

cdf_unx        HP-UX (Unix) binary executable file for CDF version
               1.1.

cdf_lnx        Red Hat Linux binary executable file for CDF version
               1.1.

cdf.f          Fortran source code for CDF version 1.1.

dcdflib11.f    Fortran source for CDF Library version 1.1.

cdf.txt        Documentation for CDF version 1.1.
